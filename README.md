# 🧮 Modern Multi-Theme Calculator

A modern, responsive calculator built using **HTML, CSS, and JavaScript**, featuring multiple UI themes, history tracking, and clean logic implementation.

🔗 **Live Demo:**  
https://hruthyacv.github.io/python-calculator/

---

## 🚀 Features

### 🎨 Theme System
- 🌸 Spring (Light pastel UI)
- ❄️ Winter (Dark mode UI)
- 💜 Lavender (Soft aesthetic theme)
- Theme selection UI
- Persistent theme (saved using localStorage)

### 🧠 Calculator Logic
- Addition, Subtraction, Multiplication, Division
- Input validation (prevents invalid inputs)

### 📜 History Panel
- Stores previous calculations
- Displays full expressions
- Updates dynamically

### 🎯 UI/UX
- Rounded buttons and soft shadows
- Smooth button press animations
- Clean and minimal design

---

## 🛠️ Tech Stack

- **HTML**
- **CSS**
- **JavaScript**
